# Guide Content Instructions

Use these rules whenever creating or revising content inside `guide/`.

## Primary source preference

1. Use the user-provided exam outline as the structure and coverage checklist.
2. Use the NCBI / Open RN **Nursing Fundamentals** book as the **preferred content source whenever it covers the topic**:
   - https://www.ncbi.nlm.nih.gov/books/NBK591823/
3. Use other reliable nursing or evidence-based sources only to:
   - fill gaps not covered well in the NCBI book
   - strengthen exam-style angles such as priority, safety, red flags, compare/contrast, or clinical judgment

## Style rules

- Keep the language very clear and friendly for a non-native English speaker.
- Do not use overly sophisticated wording.
- Aim for the sweet spot between a focused guide and an easy guide: very friendly, very clear, but still fully accurate.
- Match the original outline headings as closely as possible so progress is easy to track.
- Add short explanation lines only when they truly help.
- Under technical or less familiar subheadings, add one short plain-English helper line in parentheses on the next line when it makes the topic easier to understand.
- Do **not** put explanation text on the same line as the heading.
- Expand abbreviations in plain English when helpful, then include the abbreviation in parentheses.
- Prefer short bullets, simple sentences, and high-yield nursing points.
- Do **not** mention source names such as Open RN inside learner-facing guides or flashcards unless the user specifically asks for source discussion.
- When possible, say the plain-English idea first and then the nursing term, such as: `SA node: starts the heartbeat. It is the heart's natural pacemaker.`

## File conventions

- Group content by exam in subfolders such as:
  - `guide/exam-1/`
  - `guide/exam-2/`
- For standalone topic packs that are not tied to a numbered exam, use a short topic folder such as:
  - `guide/cardio/`
  - `guide/fluids/`
- For each exam, prefer these files:
  - one main guide named like `exam-1-guide.md` or `exam-2-guide.md`
  - flashcards
- For standalone topic packs, prefer these files:
  - one main guide named like `cardio-guide.md` or `fluids-guide.md`
  - matching flashcards named like `cardio-flashcards.md` or `fluids-flashcards.md`
- Do not keep a separate easy-memory guide. Put the clear, simple wording directly into the main guide.

## Flashcard rules

- Group flashcards by the same major outline sections.
- Include a mix of:
  - recall cards
  - compare/contrast cards
  - priority cards
  - safety cards
  - application or scenario cards
- Keep each flashcard answer tight and testable.
- Prefer strong nursing-exam angles:
  - ABC priority
  - Maslow priority
  - safety first
  - early recognition of complications
  - client teaching

## Accuracy rules

- Prefer the NCBI / Open RN textbook wording and concepts whenever possible.
- If another source is used, make sure it does not conflict with the NCBI / Open RN content.
- When unsure, favor standard nursing-fundamentals teaching rather than niche or advanced specialty detail.

## Revision rules

- When revising an existing file, improve it without dropping outline coverage.
- Preserve simple wording and user-friendly formatting.
- Keep the final result useful for both quick review and exam prep.
